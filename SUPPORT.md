# Support — LoopTidy

LoopTidy is a local-first open-loop tracker for follow-ups, blockers, commitments, and decisions.

## Contact

**Email (preferred):** hello.hailelabs@gmail.com

**GitHub issues:** https://github.com/haile-y-haile/Looptidy/issues

## What to include

- Device model (e.g. iPhone 15)
- iOS version (e.g. iOS 18.4)
- App version and build number (Settings → About)
- Steps to reproduce the issue
- Screenshot if helpful (redact sensitive loop content)

## Privacy

https://haile-y-haile.github.io/Looptidy/privacy

## Current version behavior

- All loop data stays on your device (SQLite)
- Local reminders use on-device notifications only
- No account sign-in required
- Optional Face ID app lock in Settings
- Backup & Restore for export/import on this device

## Security

Do not post passwords or exploit details in public issues. See [SECURITY.md](SECURITY.md).
